# Criação colunas
## CLASSIFICACAO_CORRETOR
- Gerar classificação para o corretor
- Coluna será categórica
- Classificação do corretor respeitará segmentação (Exemplo: Manual/Digital; Novo/Renovação; Produto; etc)
- Utilizar diversos campos da tabela cotacao_generico e criar a classificação do corretor, por exemplo
    - HR Alto / QTD_COTACOES Alto / Generalista / Alta tendência de crescimento
    - HR Baixo / QTD_COTACOES Baixo / Especialista / Baixa tendência de crescimento
